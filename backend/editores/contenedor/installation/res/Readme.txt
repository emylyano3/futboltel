Futboltel Admin
===============

Es una aplicacion que permite tanto la administracion de usuarios, perfiles y permisos como la edicion del contenido habilitado para los consumidores.
Esta conformada por 2 modulos:
1 Modulo Adminstracion
	Permite la adminsitracion y gestion de usuarios, perfiles, accesos y modulos. Sirve como herramienta de administacion se seguridad.
2 Modulo Contenido
	Permite la administracion de datos de consumo.
